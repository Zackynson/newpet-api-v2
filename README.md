# newpet-api-v2 🐕
Nova versão da api do newpet. Criada usando TDD, conceitos de SOLID e Typescript. 

## HOW TO INSTALL ⚙️
* npm install

## HOW TO TEST 🧪
#### UNIT: 
* npm run test:unit

#### INTEGRATION:
* npm run test:integration

#### BOTH:
npm run test

#### GENERATE COVERAGE REPORT: ✔️
npm run coverage

## HOW TO RUN 🚀
* npm run dev
